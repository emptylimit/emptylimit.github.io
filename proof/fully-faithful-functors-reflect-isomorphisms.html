Fully Faithful Functors Reflect Isomorphisms

Written by: Gabriel Field.

Theorem statement

Theorem.

Let $F : \mathsf{C} \to \mathsf{D}$ be a functor and let $x \xrightarrow{i} y$ be an arrow in $\mathsf{C}$. Suppose that $F$ is fully faithful and $F i : F x \simeq F y$ is an isomorphism in $\mathsf{D}$. Then, $i : x \simeq y$ is an isomorphism in $\mathsf{C}$.

Direct Proof

We next prove the main result. The isomorphism $F i : F x \simeq F y$ has an inverse $\left(F i\right)^{-1} \in \mathsf{D}(F y, F x)$. $F : \mathsf{C}(y, x) \to \mathsf{D}(F y, F x)$ is surjective (because $F$ is full), so there exists $j \in \mathsf{C}(y, x)$ with $F j = \left(F i\right)^{-1}$. By injectivity of $F : \mathsf{C}(x, x) \to \mathsf{D}(F x, F x)$, establishing $1_x = j \circ i$ is equivalent to establishing $F 1_x = F (j \circ i)$. Indeed, $$\begin{align*} F 1_x = 1_{F x} = F j \circ F i = F (j \circ i) \end{align*}$$ and similarly, $1_y = i \circ j$. Thus, $i$ has inverse $j$.

Alternative Proof

$F$ being fully faithful asserts that for all $a, b \in \mathsf{C}$, the function $F : \mathsf{C}(a, b) \simeq \mathsf{D}(F a, F b)$ is an isomorphism. In essence, it says that $\mathsf{C}$ is "the same" as those parts of $\mathsf{D}$ whose objects lie in the image of $F$.

Let's make this precise. Let $\mathsf{E}$ be the full subcategory of $\mathsf{D}$ spanned by collection of objects $\left\{F c \;\middle\vert\; c \in \mathsf{C} \right\}$. Construct a functor $F^{-1} : \mathsf{E} \to \mathsf{C}$ as follows. For each object $\tilde{x} \in \mathsf{E}$, choose an object $c \in \mathsf{C}$ such that $F c = \tilde{x}$, and assign $F^{-1} (\tilde{x}) := c$. In particular, choose $F^{-1} (F x) := x$ and $F^{-1} (F y) := y$. Now, define the maps $ F^{-1} : \mathsf{E}(\tilde{x}, \tilde{y}) = \mathsf{D}( F\left( F^{-1} \tilde{x} \right), F\left( F^{-1} \tilde{x} \right) ) \to \mathsf{C}(F^{-1} \tilde{x}, F^{-1} \tilde{y}) $ to be the inverse functions to each $ F : \mathsf{C}(F^{-1} \tilde{x}, F^{-1} \tilde{y}) \to \mathsf{D}( F\left( F^{-1} \tilde{x} \right), F\left( F^{-1} \tilde{x} \right) ) = \mathsf{E}(\tilde{x}, \tilde{y}) $. It is easy to verify that $F^{-1}$ is a functor; in fact, it is a special case of the inverse-equivalence functor constructed in our proof that fully faithful and essentially surjective functors are equivalences of categories.

The main proof is now easy. $F^{-1}$ is a functor, and functors preserve isomorphisms, so $F^{-1}$ maps the isomorphism $F i$ to an isomorphism in $\mathsf{C}$. By construction, this isomorphism is $i$.

This proof underscores the "real meat" of what is happening here. The fully fathful functor $F : \mathsf{C} \to \mathsf{D}$ is like an injective function $f : C \to D$; like an injective function is a bijection onto its image $f : C \simeq \mathrm{Im}(f) \subseteq D$, a fully faithful functor defines an equivalence onto its image $F : C \simeq \mathsf{E}$. Equivalent categories exhibit the same categorical features, so we should expect the isomorphism $F i$ in the image of $F$ to correspond to an isomorphism $i$.

Example Uses

  • The Yoneda Embedding is Fully Faithful, which enables "equational style" proofs of isomorphisms, relating them to isomorphisms in $\mathbf{Set}$.
  • Any equivalence of categories is fully faithful. Hence, one can identify isomorphisms in one category by recognising isomorphisms in an equivalent category. For example, since the category of finite-dimensional vector spaces is equivalent to the category of matrices (via a "matrix representation" functor), one can identify isomorphisms of vector spaces by recognising invertible matrices, which can often be done by computing determinants.
