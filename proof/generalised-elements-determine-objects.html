Generalised Elements Determine Objects

Written by: Gabriel Field.

Theorem statement

Theorem.

Fix a locally small category $\mathsf{C}$ and objects $x, y \in \mathsf{C}$. Then, $x \simeq y$ in $\mathsf{C}$ if and only if the functors $\mathsf{C}({-}, x), \mathsf{C}({-}, y) : \mathsf{C}^{\mathrm{op}} \to \mathbf{Set}$ are naturally isomorphic.

Dually, $x \simeq y$ in $\mathsf{C}$ if and only if $\mathsf{C}(x, {-}) \simeq \mathsf{C}(y, {-})$.

This result is often also referred to as the Yoneda lemma, despite the Yoneda lemma being a more general statement.

Proof

The Yoneda embedding is the functor $よ : \mathsf{C} \to \mathsf{C}at(\mathsf{C}^{\mathrm{op}}, \mathbf{Set})$ given by $ \left( c \xrightarrow{f} d \right) \xmapsto{よ} \left( \mathsf{C}({-}, c) \xrightarrow{f \circ {-}} \mathsf{C}({-}, d) \right) $. Functors preserve isomorphisms, so if $c \simeq d$, then $よ\ c \simeq よ\ d$. Furthermore, the Yoneda embedding is fully faithful and fully faithful functors reflect isomorphisms, so $よ\ c \simeq よ\ d$ implies $c \simeq d$.
